2. Вывести список кодов продавцов в обратном порядке.

3. Вывести все фамилии и товар заказчиков в обратном порядке (сортировка по фамилии), если фамилия входит в диапазон [С-Л].
	SELECT NAME, Prod FROM ZAKAZ WHERE NAME GLOB '[Л-С]*' ORDER BY NAME DESC
	SELECT NAME, Prod FROM ZAKAZ WHERE NAME GLOB '[Л-С]*' ORDER BY 1 DESC
	


4. Вывести фамилии заказчиков в интервале от Па до Ле, которые не продают никакие модели  шкафов.
	SELECT NAME, ProdFROM ZAKAZWHERE NAME BETWEEN 'Ле*' AND 'Па*' AND Prod NOT LIKE 		'шкафы'
	SELECT * FROM ZAKAZ WHERE NAME BETWEEN 'Ле' AND "Пб" AND Prod != 'шкафы'  ORDER 		BY NAME DESC;


5. Выведите фамилии продавцов, сумма заказа которых превышает 4000 и код продавцов 1003, 1010 или 1016 (три способа).
	SELECT * FROM ZAKAZ WHERE SUM >4000 AND (KOD == 1003 OR KOD == 1010 OR KOD == 		1016)
	SELECT * FROM ZAKAZ WHERE SUM >4000 AND KOD IN (1003, 1010, 1016)
